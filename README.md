# Georgia Elections Data Dashboard

This repository contains a dashboard for visualizing and analyzing election data from the state of Georgia. The dashboard documents official election results, voter turnout, and candidates who participated in elections held in Georgia.

## Technologies Used

- Observable Framework for building the dashboard
- D3.js for data visualization
- HTML/CSS for structuring and styling the dashboard
- JavaScript for interactivity and data manipulation
- Git for version control
- R (through OFHQ data loaders) for data cleaning and preprocessing
- LLMs for assistance with coding and data analysis (ChatGPT, Google Gemini)
